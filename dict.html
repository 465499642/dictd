dict project

dict is my programming project to satisfy the CS 490 project requirement for my BS degree in Yale's Computer Science major. Professor Zhong Shao is my project advisor.

Read my original project proposal for a more in-depth description.

Note: reverse matching can take a long time depending on the length of your search string. Because of the size of the Webster database, reverse matching is not yet implemented for it.
Here are some sample queries, just to give you an idea of what you can do.
  • All the words starting with "pre" (query: "pre", prefix match, any database)
  • All the words containing the string "cow" (query: "cow", substring match, any database)
  • All the kinds of SCSI according to the Free Online Dictionary of Computing (query: "SCSI", substring match, FOLDOC)
Grab the current Perl DICT system client to have a network dictionary at your UNIX command-line.
